### 문제 설명

<p>整数 n (2 ≤ n ≤ 100000000) が与えられる. m の階乗が n で割り切れるような最小の正の整 数 m を求めるプログラムを書け. ただし, 正の整数 m に対し, m の階乗とは 1 から m までの 整数の積のことである.</p>

### 입력 

 <p>このファイルは 1 行からなり, 整数 n が書かれている.</p>

### 출력 

 <p>プログラムは結果を標準出力に出力すること. 整数 m のみを書いた 1 行を出力せよ.</p>
